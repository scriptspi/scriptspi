Como funciona?

Funciona da seguinte forma, você deve abrir a página headless host e colocar o seu script lá, siga as instruções abaixo.

  • Abra a página headless host Headless Host
  • Aperte a tecla F12
  • Dentro do console cole todo o seu script no console.
  • Pronto siga o link que o console te deu para a sua sala com bot.
